# Cooldown Overlay

A Windows-only external cooldown overlay. It observes physical number-row skill keys and MB1 without reading game memory, injecting into a game, or automating input.
The listener uses high-resolution 1 ms input checks and Windows' press-transition signal to retain very fast taps and clicks.

# How to Use
You MUST have the newest version of python which you can get at 'https://www.python.org'  Specifically get version 3.1 or higher
